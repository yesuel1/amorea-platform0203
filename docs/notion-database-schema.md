## 🔗 노션 API 연동 가이드

### 1단계: 노션 통합 생성
1. https://www.notion.so/my-integrations 접속
2. "New integration" 클릭
3. 이름: "AMOREA CMS"
4. Associated workspace 선택
5. Capabilities: Read content, Update content, Insert content
6. **Integration Token 복사** (나중에 필요)

### 2단계: 데이터베이스 생성 & 공유
1. 노션에서 새 페이지 생성
2. 위 스키마대로 각 데이터베이스 생성
3. 각 데이터베이스 우측 상단 "···" → "Add connections" → AMOREA CMS 선택

### 3단계: Database ID 가져오기
- 데이터베이스 URL: `https://notion.so/workspace/{database_id}?v=...`
- `database_id` 부분 복사
